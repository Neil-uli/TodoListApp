## TodoApp
Endpoints exposed

|  Endpoint   | Method |     Description     |
| :---------: | :----: | :-----------------: |
| /v1/task/id |  Get   | Get a task using ID |
|  /v1/task   |  Post  |  Create a new task  |
|  /v1/tasks  |  Get   |    Get all tasks    |
| /v1/task/id |  Put   |    Update a task    |
